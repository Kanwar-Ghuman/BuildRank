# BuildRank

AI landing page teardown for founders.

## Setup

```bash
npm install
npm run dev
```

Optional: Copy `.env.example` to `.env.local` and add Clerk keys for production auth. Keyless mode works for development.

## Project structure

- `docs/PRD.md` — Product requirements
- `docs/MILESTONES.md` — Commit milestones
- `public/logo.svg` — Replace with your logo

## Stack

Next.js 16, TypeScript, Tailwind v4, shadcn/ui, Clerk
